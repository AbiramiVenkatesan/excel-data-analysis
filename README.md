# excel-data-analysis
Data Analysis and Reporting using Microsoft Excel

Vrinda Store Annual Sales Analysis
Microsoft Excel | Data Analytics Portfolio Project

Project Overview
This project focuses on analyzing the annual sales data of Vrinda Store to uncover critical insights regarding customer demographics, purchasing behavior, regional performance, and sales channels. The ultimate goal is to deliver data-driven recommendations to improve sales strategies for the upcoming year.

Business Questions Answered
**Sales vs. Orders:** How do monthly sales trends compare directly to order volumes?
**Peak Performance:** Which month generated the highest revenue and order traffic?
**Gender Demographics:** Who purchases more between men and women?
**Order Fullfillment:** What is the structural breakdown of order statuses (Delivered, Cancelled, Returned)?
**Geographic Distribution:** Which top 5 states contribute the most to overall revenue?
**Target Audience:** What is the relationship between buyer age groups and gender?
**Channel Distribution:** Which external e-commerce channel drives the maximum sales volume?

Data Analysis Process
1. **Data Cleaning:** Verified datasets for missing values, handled anomalies, and ensured strict data type consistency.
2. **Feature Engineering:** Created custom `Age Group` logic (`Adult`, `Senior`, `Teenager`) using conditional formulas to better segment the customer profile.
3. **Data Aggregation:** Built 6 distinct Pivot Tables to segment metrics across geography, demographics, and active sales channels.
4. **Interactive Dashboarding:** Combined all visual charts into a consolidated reporting sheet, utilizing dynamic **Slicers** to enable seamless filtration by month and category.

Key Insights & Business Recommendations
* **Demographic Focus:** Women represent approximately **64%** of total purchases, with the **Adult age group (30-49 years)** serving as the highest-performing consumer segment.
* **Geographic Hubs:** **Maharashtra, Karnataka, and Uttar Pradesh** represent the top 3 revenue-generating states.
* **Platform Performance:** Core marketplace platforms (**Amazon, Flipkart, and Myntra**) collectively drive nearly **80%** of all inbound store orders.
* **Actionable Strategy:** Target adult women (aged 30-49) in Maharashtra, Karnataka, and UP by focusing marketing spend and promotional offers directly through Amazon, Flipkart, and Myntra storefronts.
